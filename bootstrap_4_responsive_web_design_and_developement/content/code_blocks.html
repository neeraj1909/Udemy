<p>Use the <code> element wrapped in a <pre> element to write code blocks.</p>
    <p>Adding the class .pre-scrollable will set a max-height of 350px and provide a vertical scrollbar.</p>
    <p>Variables should be wrapped in a <var> element like this: God = true.</p>
    <p>Keyboard shortcuts should be wrapped in a <kbd> element like this: Esc and this: Cmd + S.</p>
    <p>Sample output from a program should be wrapped in a <samp> element like this:</p>
    This is a sample output from a program.
